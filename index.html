Proyek Unggulan

Aplikasi Web

Sistem Manajemen Sekolah

Aplikasi web berbasis PHP, HTML, CSS, dan JavaScript untuk pengelolaan data siswa, guru, jadwal pelajaran, dan nilai siswa-siswi di SMA Unggul Del secara efisien. Program ini menggunakan database MySQL untuk penyimpanan data dan memiliki tiga role utama yaitu admin, guru, dan siswa. Admin dapat mengelola data siswa, guru, dan jadwal pelajaran, guru memiliki fitur input nilai dan absensi, sedangkan siswa dapat melihat nilai serta jadwal pelajaran mereka.

Aplikasi Mobile

"Agenda Acara" Mobile App

Aplikasi mobile Agenda Acara yang dikembangkan menggunakan Flutter dan Firebase. Aplikasi ini memungkinkan pengguna untuk membuat, mengededit, menstruktur acara yang akan datang dengan mudah. Fitur utama nya adalah ada nya history acara yang telah selesai dilakukan sehingga pengguna dapat melihat kembali acara acara yang telah dilakukan sebelumnya. Aplikasi ini juga terintegrasi dengan API Firebase untuk penyimpanan data secara real-time dan autentikasi pengguna.

Aplikasi Desktop

Aplikasi Pembayaran SPP

Aplikasi Pembayaran SPP yang melakukan pencatatan pembayran siswa siswa di SMA N 1 Silaen. Aplikasi akan menyimpan nama nama siswa yang sudah selesai melakukan pembayaran dalam rentang 1 tahun/ 12 bulan. Bagi siswa yang belum melakukan pembayaran dalam rentang waktu tersebut akan diberikan status menunggak dan bagi yang sudah akan ada keterangan lunas.
